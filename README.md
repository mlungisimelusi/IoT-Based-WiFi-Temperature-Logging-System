# IoT-Based WiFi Temperature Logging System

A compact and efficient **IoT-based wireless system** designed for **real-time temperature and humidity monitoring**.  
This project uses WiFi-enabled microcontrollers to **collect, transmit, and log environmental data**, making it useful for home automation, greenhouses, storage monitoring, research labs, and more.

- **Lightweight & Efficient:** Implemented primarily in **C++**, optimized for microcontrollers.  
- **Scalable IoT Design:** Multiple sensor nodes can report to a single server.
